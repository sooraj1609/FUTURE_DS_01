# Global Retail Sales Performance Dashboard

## Project Overview
This project analyzes retail sales data using Tableau to uncover revenue trends, customer purchasing behavior, and product performance insights. The dashboard is designed to help businesses make data-driven decisions using interactive visual analytics.

---

## Objectives
- Analyze monthly revenue trends
- Identify top-performing products
- Understand customer contribution to revenue
- Discover product demand patterns
- Build an interactive executive dashboard

---

## Tools Used
- Tableau
- Excel / CSV
- GitHub

---

## Dashboard Features
- KPI Cards
- Monthly Revenue Trend Analysis
- Top Products by Revenue
- Top Customers Analysis
- Product Demand Scatter Plot
- Interactive Dashboard Filters

---

## Key Business Insights
- Revenue showed noticeable seasonal fluctuations across months.
- A limited number of products contributed significantly to total revenue.
- High-value customers generated a major share of overall sales.
- Certain products achieved high sales quantities but relatively lower revenue.
