## Project Overview

This project analyzes website traffic and marketing performance using Google Analytics data.

### Tools Used

- Tableau Public
- Python
- Pandas
- VS Code

### Key Insights

- Organic Search generated the highest traffic.
- Google and YouTube were the top acquisition sources.
- Desktop users contributed most sessions.
- United States generated the highest traffic.
- Bounce rates varied significantly across channels.

### Recommendations

- Increase investment in high-performing channels.
- Optimize high-bounce landing pages.
- Improve mobile engagement.
- Expand campaigns in top-performing regions.
