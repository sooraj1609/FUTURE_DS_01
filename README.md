- Interactive KPI dashboard

### Task 3 – Marketing Funnel & Conversion Performance Analysis

A marketing analytics dashboard developed using Google Analytics data to evaluate traffic sources, user engagement, bounce behavior, and conversion performance.

- Channel performance comparison

- Geographic and device insights
